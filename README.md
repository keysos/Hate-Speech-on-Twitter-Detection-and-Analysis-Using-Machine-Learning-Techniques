# Discurso-de-dio-no-Twitter-Detec-o-e-An-lise-por-Meio-de-T-cnicas-de-Machine-Learning.
Este repositório reúne os materiais, códigos e experimentos desenvolvidos em uma pesquisa acadêmica realizada na Universidade Federal de Sergipe (UFS), cujo objetivo é analisar e comparar diferentes técnicas de Machine Learning aplicadas à detecção automática de discurso de ódio na rede social Twitter.
